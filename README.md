# TrustCore CLI - Contest Edition 🌿

A command-line interface (CLI) tool to calculate a "TrustScore" based on user blockchain interactions.  
This project was built for the **Canopy Vibe Coding Contest** using TypeScript, Node.js, and AI-inspired design concepts.

---

## Features

- Input multiple blockchain entries (transactions, smart contracts, reputation score).  
- Computes a TrustScore for each entry using a simple formula.  
- Saves results to `trustcore-results.json`.  
- Logs activity to `trustcore-demo.log` for easy demo/video recording.  
- Minimal setup and cross-platform (Windows/Linux/macOS).
